IBIS System Version A

Introduction

This is the IBIS System Version A, the intentions of this system is to simplify controlling various devices with the goal of easy or no programming skills required to do so.

The GitHub location for the original code is https://github.com/dan157/PTC and the YouTube channel used to watch the videos to explain various features of this system can be found searching for the channel called "Pacific Tech Center".

Installation

Step 1: Clone the github page https://github.com/dan157/PTC

Step 2: Compile the program by typing "make" from the PTC/IBISvA directory

Step 3: Read the manual, currently located in the PTC/IBISvA/Examples/Chapter1, the UserManual.pdf or UserManual.docx should exists if the program was cloned fully

Step 4: Either copy the system.xml file from the same directory, or create a new one and place it into the /etc/IBIS/conf/ directory

Step 5: Either copy the example *.xml files (all except for the system.xml file) to the /etc/IBIS/conf.d/ directory or create your own State Machine program XML files and place them here

Step 6: Run the IBIS program "./IBISvA" or redirect to /dev/null if you don't want to see the debug code "./IBISvA > /dev/null"
